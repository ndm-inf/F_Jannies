About BlockChan, Our Rules, and Posting Anonymously

Wait, what is this creepy place?

We are a 100% free, censorship resistant, immutable, open source decentralized image board. Our code repository is here: https://github.com/ndm-inf/BlockChan

With BlockChan there is no central server hosting the content and no web server logs to worry about. BlockChan is a client interacting with the Ripple ledger/blockhain (for text content) and IPFS (for media). Anyone is free to install the client which is distributed for free on our github. You can also fork us and alter your client as you see fit. You can even publish it online under your own domain.

Anonymously uploading to BlockChan

Whistleblower looking to anonymously share something? We got you.

To upload a file using BlockChan to the Ripple blockchain anonymously, you must hide your identity from two servers. The Ripple Ledger that your post is submitted to and and the IPFS server the image is pinned to (Pinata).

Anonymously connecting to Ripple Decentralized Network and IPFS

The issue to address is anonymously and securely connecting to a server instance of the ripple blockchain and Pinata (the IPFS API BlockChan uses). Do the following:

First install Brave Browser

First, install the Brave Browser. This browser offers cutting edge privacy and security with the ability to use Tor which furthers obfuscates the ability for a server to know the actual source of it's client. MAKE SURE you are in a Tor window before proceeding (ALT-SHIFT-N).

Obtain and Setup VPN (Suggested: ProtonMail + ProtonVPN)

Secondly, while using Brave in a TOR window, you will want to setup a VPN, we suggest using a Proton Email address, and ProtonVPN. Email setup is very straight forward and the VPN setup has directions that are well laid out and easy to follow. It is highly advised to upgrade your free account to a paid account which will give you the ability to choose which country your VPN routes through. If you do decide to upgrade, do not pay with a debit card or paypal, but instead use bitcoin. You can follow the steps in the below section on how to Anonymize your bitcoin (or any crypto for that matter).

Burner Phone + Tether

Lastly, when ready to post w/ BlockChan, for further security, you can use a Burner phone w/ tether internet capability that you purchased with cash as your internet source.

There are other privacy browsers w/ tor capabilities that are considered to be better, however Brave is the easiest. There are also other VPN's which are considered more secure. DYOR and due diligence. If you have any suggestions to improve this document, please get in touch with us.
